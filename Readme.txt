실행하면

윈도우 창 File 안에 start를 누르면 게임이 시작한다

처음엔 정비 상태로 게임이 시작하며 add wall, sell mod, add tower, upgrade tower 버튼을 누르면 클릭 상태가 변하여

맵을 마우스로 클릭하면 벽을 세우거나, 포탑을 세우거나, 업그레이드, 철거 등을 할 수 있다
만약 벽을 설치 할 수 없는 곳에 클릭 등 할 수 없는 행위를 할 때 터미널 창에 실패 등의 문구가 출력된다

round start 버튼을 누르면 적이 등장하며 이때는 클릭 모드를 사용할 수 없다(벽을 세우거나 등등)

round가 시작되면 적이 등장하며 적은 거점으로 이동하고 포탑은 적을 공격한다
이때 포탑은 공격가능 상태일때 투명도가 진하며 쿨타임일때는 반투명하다

hp가 0이하가 되면 game이 끝나고 round를 15까지 가면 game이 승리한다


게임을 다시 시작하고 싶다면 file에 start 버튼을 눌러야한다
